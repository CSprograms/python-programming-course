# Python Programming Course - UCAM11I

BCA Section 1, Sri Sankara Arts and Science College (Autonomous), Enathur.

This repository contains the runnable Python programs for the 75-session
Python Programming course, organized session by session.

## Contents

- **[`Session_Programs/`](Session_Programs/)** - one folder per session
  (`Session_01` ... `Session_75`), each with the Session Programs and Workout
  Programs demonstrated/assigned that hour. See
  [`Session_Programs/README.md`](Session_Programs/README.md) for the full
  layout (teaching sessions, assignments, class tests, seminar/discussion
  sessions).
- **[`Supplementary-NCERT-Programs/`](Supplementary-NCERT-Programs/)** - 92
  extra practice programs from the NCERT-aligned Computer Science /
  Informatics Practices coursework (Classes XI-XII), organized `Unit_01`
  ... `Unit_11` to mirror the course units. Supplementary to the session
  programs above, not part of the graded session plan. See
  [`Supplementary-NCERT-Programs/README.md`](Supplementary-NCERT-Programs/README.md).

## Course units

| Unit | Title                          | Sessions |
|------|--------------------------------|----------|
| I    | Introduction to Python         | 1-15     |
| II   | Flow Control                   | 16-30    |
| III  | Functions and Exception Handling | 31-45  |
| IV   | Strings and Lists              | 46-60    |
| V    | Tuples and Dictionaries         | 61-75    |

## Running a program

Each file under `Session_Programs/` is a standalone script:

```
python Session_Programs/Session_01/Program_1_Hello_Python.py
```

Every file's docstring documents the program's description and its sample
output, so you can check your own run against the expected result.
